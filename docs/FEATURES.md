# Features List (v2)

## 1. Authentication (Hybrid)
- Local Email/Password registration and login.
- Google OAuth login.

## 2. Hierarchical File Upload
- Users must navigate a strict hierarchy to upload notes: University -> Course -> Semester -> Subject -> Resource Type.
- Files and Images are stored remotely in Cloudinary.

## 3. View-Only Dashboard
- The main feed is public.
- Engaging with posts (likes, comments, messaging) requires authentication.

## 4. Moderation & Reporting
- Built-in reporting tool to flag inappropriate resources or users.
